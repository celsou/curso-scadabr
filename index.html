Perguntas frequentes

O que é o ScadaBR?

O ScadaBR é um software livre o open-source para criação de sistemas supervisórios do tipo SCADA (Supervisory Control and Data Acquisiton). Ele foi desenvolvido por uma comunidade brasileira e pode ser baixado pelo site scadabr.com.br

O que é o Curso ScadaBR 1.2 do canal Celso Automação?

O Curso ScadaBR 1.2 é um curso gratuito e completo sobre o software ScadaBR em sua versão mais recente (1.2). Este curso aborda de forma didática os recursos do ScadaBR, como instalá-lo e utilizá-lo.

Quanto custa o Curso ScadaBR 1.2?

O Curso ScadaBR 1.2 é completamente gratuito, estando as aulas disponíveis neste site e no canal do YouTube Celso Automação.

Este curso serve para o Scada-LTS?

Na maioria dos casos, sim. Embora o foco deste curso seja o ScadaBR, como a interface clássica do Scada-LTS é praticamente idêntica e ambos são projetos irmãos, a maioria dos conteúdos deste curso podem ser aplicados ao Scada-LTS.

Qual a diferença entre ScadaBR e Scada-LTS?

O Scada-LTS surgiu como uma ramificação (fork) do ScadaBR, visando trazer novos recursos e acelerar o desenvolvimento do software. Enquanto o ScadaBR possui desenvolvimento mais lento e está limitado (atualmente) à comunidade brasileira, o Scada-LTS é um projeto internacional, com novas versões sendo lançadas com frequência. Ambos os projetos são maduros e podem ser utilizados em ambientes de produção.

Este curso possui certificado de conclusão?

Não, como o foco deste curso não é comercial, não oferecemos certificados de conclusão.

Este curso é voltado à formação de integradores do ScadaBR?

Não. Este é um curso focado em dar uma visão geral sobre os recursos do software ScadaBR, não sendo voltado a formar profissionais integradores do ScadaBR.

Se você precisa aprender a utilizar o ScadaBR para uso profissional, e tornar-se um integrador certificado, confira o curso oficial de ScadaBR/Scada-LTS .
